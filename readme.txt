Smart Scroll to Top adds a sleek and customizable 'Back to Top' button with smooth scrolling to your WordPress site.

== Description ==
Simple Scroll to Top is a lightweight and user-friendly WordPress plugin that adds a beautiful 'Back to Top' button to your website. It helps improve user experience by allowing visitors to quickly return to the top of the page with a smooth scrolling animation.

Key Features:
* Smooth scrolling animation
* Customizable button appearance
* Mobile-friendly design
* Easy to use and configure
* Lightweight and fast loading
* Works with all WordPress themes

== Installation ==
1. Upload `simple-scroll-top-wp` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Appearance > Customize > Scroll To Top" to configure the button settings

== Frequently asked questions ==
= Do I need to setup anything? =
No, the plugin works out of the box. However, you can customize the button appearance through the WordPress Customizer.

= Is this plugin mobile-friendly? =
Yes, the plugin is fully responsive and works perfectly on all devices.

= Will this plugin slow down my website? =
No, the plugin is lightweight and optimized for performance.
